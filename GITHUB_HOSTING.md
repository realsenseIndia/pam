# Hosting PAM on GitHub (Android Setup)

Follow these steps to get a professional, installable link for your phone.

## Step 1: Create a New Repository
1.  Log in to your GitHub account (`realsenseindia@yahoo.com`).
2.  Click the **"+"** icon in the top-right and select **"New repository"**.
3.  **Repository name**: `pam`
4.  Set it to **Public**.
5.  Click **"Create repository"**.

## Step 2: Upload your Files
1.  On the next screen, click the link that says **"uploading an existing file"**.
2.  Open your computer folder: `c:\Users\hp\My Gravity Projects\Personal Accounts Manger`
3.  **Drag and drop** ALL files and folders from that folder into the GitHub upload area:
    *   `css` (folder)
    *   `js` (folder)
    *   `index.html`
    *   `manifest.json`
    *   `sw.js`
    *   `icon-512.png`
4.  Wait for all files to upload.
5.  Click **"Commit changes"** at the bottom.

## Step 3: Enable GitHub Pages
1.  Go to the **"Settings"** tab of your new `pam` repository.
2.  In the left sidebar, click **"Pages"**.
3.  Under **"Build and deployment"** -> **"Branch"**:
    *   Select `main` (or `master`).
    *   Select `/ (root)`.
4.  Click **"Save"**.

## Step 4: Access on Android
1.  Wait about 1-2 minutes for GitHub to build the site.
2.  Your link will be: `https://realsenseindia.github.io/pam/` (Note: `realsenseindia` might be your GitHub ID, if it's different, use your ID).
3.  Open this link in **Chrome** on your Android phone.

Now you have a real app icon on your phone that works everywhere!
